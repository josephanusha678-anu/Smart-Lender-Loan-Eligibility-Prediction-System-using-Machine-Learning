# Smart Lender - Loan Eligibility Prediction System using Machine Learning

## 📌 Project Description

Smart Lender is a Machine Learning-based web application developed using Python and Flask. The main objective of this project is to predict whether a loan applicant is eligible for a loan based on the applicant's income, loan amount, and credit history.

The application provides a simple and user-friendly interface where users can enter the required details. Based on the trained Machine Learning model, the application predicts whether the loan should be approved or rejected.

This project demonstrates the practical implementation of Machine Learning, Data Preprocessing, Model Training, Model Deployment, and Flask Web Development.

---

# 🚀 Working

1. User opens the application.
2. Enter Applicant Income.
3. Enter Loan Amount.
4. Enter Credit History (1 = Good, 0 = Bad).
5. Click the Predict button.
6. The Machine Learning model processes the data.
7. The application displays whether the loan is Approved or Rejected.

# 📸 Output

The system predicts one of the following:

- Loan Approved
- Loan Rejected
