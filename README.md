# Smart Lender - Loan Eligibility Prediction System

## 📌 Project Overview

Smart Lender is a Machine Learning-based web application that predicts whether a loan applicant is eligible for loan approval. The application analyzes applicant details such as gender, marital status, education, employment status, income, loan amount, credit history, and property area to generate an instant prediction.

The project is developed using Python, Flask, HTML, CSS, and Scikit-learn with a Random Forest Classifier as the final prediction model.

---

## 🎯 Project Objectives

- Predict loan eligibility using Machine Learning.
- Reduce manual loan verification time.
- Provide fast and accurate loan approval predictions.
- Develop a simple and user-friendly web application.
